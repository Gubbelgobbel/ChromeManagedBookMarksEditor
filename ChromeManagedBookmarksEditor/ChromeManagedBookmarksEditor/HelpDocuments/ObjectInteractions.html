Object Interactions

In the managed bookmarks display there are four buttons

  • .. : Goes up a directory
  • Add Folder : Adds a folder to the current directory
  • Add Url : Adds a url to the current directory
  • Remove Selected : Removes the currently selected object

You can select an object by clicking on it. The currently selected object will have a blue highlight.
See above 'Folder 1' is selected.

Right clicking in the empty area will bring up a context menu. shown below.
The context menu only has two options.

  • Rename the current directory. You can rename the root folder this way.
  • Clear all managed bookmarks data. This option is only available from the root folder

Right Clicking on a folder will allow you to rename that folder.
You can enter a folder by double clicking on it.

For urls, just add them and enter the information.
The first block is for the name that is displayed in Chrome. The second block is for the actual url. You can tab between url blocks while a url is selected.
