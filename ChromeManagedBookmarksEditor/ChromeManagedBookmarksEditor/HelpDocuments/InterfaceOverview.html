Interface Overview

I tried to keep the interface as simple as possible. Below is an image and some related text to explain most of the interface.

  1. The managed bookmarks display. Used to edit your managed bookmarks. More on this in the 'Object Interactions' section.
  2. This is the help button. I hope this makes sense, otherwise I have failed...
  3. The serialize button. This will convert the managed bookmarks display into JSON code and output it in the JSON code box.
  4. The copy button copies the text from the JSON code box
  5. The JSON code box. This is were you can load managed bookmarks JSON code. It is also where serialized managed bookmarks JSON will be outputted.
  6. The load button converts JSON code in the JSON Code Box into a Managed Bookmarks object and displays it in the managed bookmarks display
